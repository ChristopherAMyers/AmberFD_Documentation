\doxysubsection{Detailed Description}
\mbox{\hyperlink{classAmberFD}{Amber\+FD}} is an umbrella function thatcombines \mbox{\hyperlink{classFlucDens}{Fluc\+Dens}} force and \mbox{\hyperlink{classDispersionPauli}{Dispersion\+Pauli}} force settings into one accessable class. This class does not calculate forces itslf, but is instead used create and access the fluctuating density and dispersion-\/pauli system and set it\textquotesingle{}s options.

To use this class, first create an \mbox{\hyperlink{classAmberFD}{Amber\+FD}} object with the number of particles to be added to the system. The, add each particle to the system by calling \mbox{\hyperlink{classAmberFD_af82eeab5280370cf5ba94100504c74ef}{add\+\_\+particle()}}, passing in a \mbox{\hyperlink{classParticleInfo}{Particle\+Info}} object. Once all particles have been added, create the fluctuating density force and dispersion-\/pauli force by calling \mbox{\hyperlink{classAmberFD_a1e63d2b77214770a66ef3045ba70deb6}{create\+\_\+fluc\+\_\+dens\+\_\+force()}} and \mbox{\hyperlink{classAmberFD_ad969170153f89634632f8bf8c02e95b2}{create\+\_\+disp\+\_\+pauli\+\_\+force()}}, respectively. Each of these forces will then be generated, and a pointer to each force can be obtained by calling \mbox{\hyperlink{classAmberFD_a9717605c8e72884bbd73441bdb4a32af}{get\+\_\+fluc\+\_\+dens\+\_\+force()}} or \mbox{\hyperlink{classAmberFD_a864dc8f930ffa8f77f51de17f92e2dd1}{get\+\_\+disp\+\_\+pauli\+\_\+force()}}.

Once the forces have been generated, adding new particles will have no effect on the system. Calling create\+\_\+\+X\+X\+X\+\_\+force() will simply return a pointer to the already created forces. If periodic boundary conditions are desired, then call \mbox{\hyperlink{classAmberFD_ac9cabb150c48a76598692395768652b9}{set\+\_\+use\+\_\+\+P\+B\+C()}} after all particles have been added.
