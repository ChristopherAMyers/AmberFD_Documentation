This class impliments the fluctuating monopole density interaction. Similar to a fluctuating charge model, this force uses atom centered monopole densities described by slater functions for the electrons and point-\/particles for the nuclei. As such, \mbox{\hyperlink{classFlucDens}{Fluc\+Dens}} includes effects of electron overlap between atomic sites.
